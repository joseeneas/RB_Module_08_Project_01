The script `analyze_salaries.py` performs the following tasks:

- Reads the dataset files (`ds_salaries-1.csv` and `ds_salaries-2.csv`) from the `Resources` folder.
- Combines the datasets and removes duplicates.
- Analyzes and visualizes various aspects of data science salaries in the US, including:
  - Average salaries by work year.
  - Average salaries by job title.
  - Average salaries by employment type.
  - Average salaries by company size.
  - Average salaries by experience level.

The generated graphs are saved in the `Graph Images` folder.
